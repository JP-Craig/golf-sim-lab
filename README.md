# Golf Simulation Lab

A physics-based golf shot simulator built in Python.

## Features
- Projectile-based shot simulation
- Monte Carlo dispersion modeling
- Drag and lift physics
- Spin-axis curvature modeling
- Data visualization of shot patterns

## Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib
- Jupyter
